## Utilisation

Cette application est un moteur de recherche spécialisé en gastronomie.

Depuis la page principale, vous pouvez rechercher un plat, un chef cuisinier ou une cuisine typique en tapant votre requête dans la barre de recherche, puis en cliquant sur le bouton correspondant.

Vous pouvez également cliquer sur les suggestions fournies par l'auto-complétion pour gagner du temps.

Une fois la liste des résultats affichée, vous pouvez cliquer sur l'un des résultats pour accéder à une page dédiée contenant toutes les informations pertinentes.
